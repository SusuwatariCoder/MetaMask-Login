## 依赖安装
```
npm install
```

### 编译运行
```
npm run serve
```

### 编译发布
```
npm run build
```

- [x] 网络状态判断
- [x] 连接账户数量判断
- [x] 连接账户信息读取
- [x] 网络类型判断
- [x] 网络切换监听重加载
- [x] 账户余额查询
